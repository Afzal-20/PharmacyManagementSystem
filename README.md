## Project Overview
A modular, offline-first desktop application designed for pharmacies to manage inventory, sales, and reporting.

### Key Features
* **Tech Stack:** Java 21, JavaFX, SQLite.
* **Architecture:** MVC Pattern with Service-Repository Layers.
* **Capabilities:** Multi-batch tracking, thermal printing integration, and strict tax/audit logging.
